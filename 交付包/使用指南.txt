# 论文降AIGC神器 - 使用指南

## 这是什么？

一套两轮提示词方案，能把论文的AIGC检测率从80%+降到20%以下。

## 怎么用？

### 方法一：直接粘贴到AI对话（最简单，推荐新手）

1. 打开 ChatGPT / 豆包 / Kimi / DeepSeek / 通义千问 / Claude 任意一个
2. 把「第一轮提示词.txt」的全部内容粘贴进去
3. 在提示词后面，把你的论文段落粘贴进去
4. 等AI改写完，复制改写结果
5. 开一个新对话，把「第二轮提示词.txt」的全部内容粘贴进去
6. 把第一轮的改写结果粘贴在后面
7. 完成！用「检查清单.txt」自查效果

### 方法二：本地脚本批量处理（适合长论文）

1. 安装 Python 3.10+
2. 克隆 baibaiAIGC 项目：https://github.com/poleHansen/baibaiAIGC
3. 配置 API Key，运行脚本自动分块处理
4. 支持 .docx 和 .txt 格式

## 重要提醒

- 论文太长（超过5000字）必须分段处理，每段不超过850字
- 两轮提示词不能合并使用，必须按顺序分两轮执行
- 不要整篇一次性丢给AI，效果会大打折扣
- 改写后请自己检查专业术语是否被误改

## 推荐AI模型

效果排序：Claude > GPT-4o > DeepSeek > Kimi > 豆包 > 通义千问

预算有限选 DeepSeek，性价比最高。

## 文件说明

- 第一轮提示词.txt - 去书面化改写
- 第二轮提示词.txt - 清AI套话改写
- 英文版提示词.txt - 英文论文专用（单轮）
- 检查清单.txt - 改写质量自查清单
- 使用指南.txt - 本文件
