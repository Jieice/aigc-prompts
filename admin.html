📖 使用说明

  1. 注册 Supabase 免费账号
  2. 创建新项目，进入 SQL Editor
  3. 执行以下 SQL 创建表： 
    CREATE TABLE codes (
  id SERIAL PRIMARY KEY,
  code TEXT UNIQUE NOT NULL,
  used BOOLEAN DEFAULT FALSE,
  used_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
  4. 在 Settings → API 获取 URL 和 anon key
  5. 填入上方配置并保存
  6. 生成解锁码，导出后配置闲鱼自动发货
